Arrays and Hashes (Technical blog) 10/03/2014
Arrays and Hashes are very similar in that they contain sets of data.

An array is an ordered list of objects. An example is the following:

this_is_an_array = [1, "this is a string", method]

A hash is an order set of key/value pairs arranged as key => "value". Each key has an associated value. An example is the following:

this_is_a_hash = {

name => "Sean",

website => "nase00.github.io",

a_number => 15,

}
Arrays and hashes are useful for storing objects. There are many methods that can then be used to search through and manipulate both arrays and hashes.
