Arrays and Hashes (Technical blog) 10/03/2014
Arrays and Hashes are very similar in that they both contain sets of data. They each can contain strings, integers, variables, and even other arrays!

An array is an ordered list of objects. An example is the following:

this_is_an_array = [1, "this is a string", method, [1, 2, 3]]

The above array contains an integer, a string, a method, and another array.

A hash is an ordered set of key/value pairs arranged as key => "value". Each key has an asigned value. An example is the following:

races_of_the_alliance = {

humans_desc => "Unbreakable spirit",

gnome_desc => "Short and intelligent",

dwarves_desc => "Short and stout",

draenei_desc => "Blue space goats"

}

Arrays and hashes are used to store a variety of objects. There are many methods that can then be used to search through and manipulate both arrays and hashes.
