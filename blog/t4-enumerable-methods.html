Enumerable Methods (Technical blog) 10/10/2014

An enumerable, in the context of Ruby, is an object that can have a block called to it one-by-one.

The following are just a few class methods that can be used to manipulate enumerables. Each class method uses the .each method to iterate over each object while calling a block, i.e., a specific task or function.

  • Enumerable#map takes a function and applies it to each individual object, then returns an array. For example, if you called .map { |n| n + 5 } on array [1, 2, 3, 4] the method would return the array [6, 7, 8, 9].

  • Enumerable#group_by Sorts each object within an array by the defined block. The output is a Hash where each key is the block it was sorted by, and each value is the element pulled from the array. For example, to group elements in an array by their modulo of 2: [2, 4, 5, 6, 7, 8].group_by { |n| n % 2 } would return the result {0=>[2, 4, 6, 8], 1=>[5, 7]}

  • Enumerable#cycle performs a loop calling the block on each enumerable. The loop runs n times or forever if no or nil value is given. It returns nil if the loop was uninterrupted after having finished all cycles.
