The differences between Margin, Border, and Padding:
  • Margin: The space between an element and other elements around it.
    In CSS, it should look like: margin: x; or it can be set to specifically top, bottom, left, or right. So, margin-top: x; for example.

  • Border: A visual wrapping around an element. It can be set to different widths, colors, styles, and inset into an element rather than displaying outside of the element.
    In CSS, it is set using several different lines, including, but not limited to: border-width: x;, border-color: #ffffff;, and border-style: x;. More border values can be found here.

  • Padding: The space between the edge of the parent element and all elements contain within itself.
    In CSS, it should look like: padding: x; or it can be set to specifically top, bottom, left, or right. So, padding-top: x; for example.
w3schools is a great reference for values that can be assigned to different CSS properties! Also, "CSS [property]" usually brings up pages with helpful lists of tricks to apply to certain properties.
