Enumerables Cheatsheet 11/2/2014

by Sean Owiecki and Devon Estes

This sheet offers a sort of "backwards lookup" for Ruby enumerable methods. Instead of finding the method you want to use and then seeing how it works, you find what you want to do and we give you the best method to use! The Enumerable Module is included by default in the Array and Hash classes, and can be mixed into any class.

I want to... Method Example
find out if any element meets a condition .any? [1, 2, 3, 4].any? { |num| num % 2 == 0 } #=> true
find out if all elements meet a condition .all? [1, 2, 3, 4].all? { |num| num.is_a? Integer} #=> true
divide an enumerable based on a block .chunk [1, 3, 2, 4].chunk { |num| num.even? }.each { |even, array| p [even, array] } #=> [false, [1, 3]] #=> [true, [2, 4]]
run a block once for each item .map or .collect { "One" => 1, "Two" => 2, "Three" => 3 }.map { |key, value| key + " " + value.to_s } #=> ["One 1", "Two 2", "Three 3"]
find how many times something appears in an object .count [1, 2, 4, 2].count(2) #=> 2
find how many times objects meet a condition .count [1, 2, 4, 2].count{ |x| x % 2 == 0 } #=> 3
run a block on each object more than once .cycle ["a", "b", "c"].cycle(2) { |x| print x * 2 } returns aabbccaabbcc
find the longest word in an array .inject or .reduce longest = %w{ cat sheep bear }.inject do |memo, word| memo.length > word.length ? memo : word end
